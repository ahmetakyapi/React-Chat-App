## Installation

##### First, clone the project.

```bash
git clone https://github.com/ahmetakyapi/React-Chat-App.git
```
##### Second, install and start the project
```bash
yarn install
# Open the server.js file in the terminal
node server.js
#Then open the client folder in the terminal
yarn start
```
